# 💳 Card Transactions KPI Pipeline

Projeto de automação em Python para processamento de dados de transações de cartões, geração de KPIs e exportação dos resultados.

---

## 🚀 Objetivo

Automatizar o fluxo de análise de dados de transações, incluindo:

- Carregamento da base de dados  
- Tratamento dos dados  
- Cálculo de métricas (KPIs)  
- Exportação dos resultados  

---

## 📊 Funcionalidades

- 📥 Leitura de dados em CSV  
- 🧹 Limpeza e tratamento de dados  
- 📊 Cálculo de KPIs como:
  - Volume total transacionado  
  - Quantidade de transações  
  - Ticket médio  
  - Taxa de aprovação  
- 📈 Análises por categoria e método de pagamento  
- 💾 Exportação dos resultados em CSV  

---

## 🛠️ Tecnologias utilizadas

- Python  
- Pandas   
- Matplotlib  

---

## ▶️ Como executar

```bash
git clone https://github.com/izaabs/automated_kpi_pipeline.git
cd automated_kpi_pipeline
pip install pandas matplotlib
python main.py
```
